Modular Interpreter

Lexer → Parser → Evaluator
Input
Ctrl+Enter to run
Token Reference
NUMBER Decimal or integer — e.g. 10, 3.14
VAR Variable name — e.g. a, result
ASSIGN Assignment operator — =
ADD Addition — a add b
MIN Subtraction — a min b
MULT Multiplication — a mult b
DIV Division — a div b
( ) Grouping — (a add b) mult c
; Statement separator
PRINT Print a value — print a;
IF Conditional — if a gt b then res = a sub b else res = b;
GT / LT Greater / less than — a gt b, a lt b
EQ / NEQ Equal / not equal — a eq b, a neq b
GTE / LTE Greater or equal / less or equal — a gte b, a lte b
WHILE Loop- while i lt 5 do i = i add 1;
FOR Counted loop - for i = 1 to 5 do sum = sum add i;
Run a program to see output
